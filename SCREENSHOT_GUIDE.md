   - Fill out form

   - Scroll to transactions
   - Hover over cards

4. **Logout and Login** (5 sec)

   - Return to login
   - Enter credentials

**Total Duration**: ~30 seconds
**Tool**: OBS Studio, QuickTime, or built-in screen recorder

## 💡 Tips

### For Best Results:
1. Use Chrome DevTools for responsive views
2. Disable browser extensions for clean screenshots
3. Use incognito mode to avoid autofill
4. Clear browser cache before recording
5. Use consistent zoom level (100%)

### Tools:
- **Screenshot**: Windows Snipping Tool, macOS Screenshot
- **Full Page**: Browser extension (Awesome Screenshot)
- **Video**: OBS Studio, QuickTime, ShareX
- **Editing**: Paint, Preview, Photoshop
